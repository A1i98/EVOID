# Installation

## Requirements

- Python 3.13+
- uv (recommended) or pip

## Install EVOID

=== "uv (Recommended)"

    ```bash
    uv add evoid
    ```

=== "pip"

    ```bash
    pip install evoid
    ```

## Verify Installation

```bash
evo version
```
